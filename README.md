# Pangram Web UI

A sleek web interface for [Pangram Labs](https://pangramlabs.com) AI text detection API. Analyze text to determine if it was written by a human or generated by AI.

## Features

- **Real-time Analysis** - Submit text and get instant AI detection results
- **Segment Highlighting** - Interactive visualization showing which parts are human vs AI-written
- **Score Breakdown** - Detailed confidence scores for human, AI, and mixed content
- **Analysis History** - Keep track of previous analyses in the sidebar
- **Dark/Light Theme** - Toggle between themes with system preference detection
- **Dashboard Links** - Direct links to detailed Pangram Labs reports

## Prerequisites

- [Bun](https://bun.sh) runtime
- Pangram Labs API key ([get one here](https://pangramlabs.com))

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/karilaa-dev/pangram-webui.git
   cd pangram-webui
   ```

2. Set your API key:
   ```bash
   export PANGRAM_API_KEY=your_api_key_here
   ```

3. Start the server:
   ```bash
   bun run start
   ```

4. Open http://localhost:3000 in your browser

## Development

Run with hot reload:
```bash
bun run dev
```
